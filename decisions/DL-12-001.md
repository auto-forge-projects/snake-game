# DL-12-001: Docker paketleme (base image, non-root, audit) + CI iyileştirmesi

- Tarih: 2026-07-19
- Faz: 12 — CI/CD
- Mod: AUTOPILOT
- Karar: Tek-aşama `node:22-alpine` imaj (`Dockerfile`); `npm ci --omit=dev` ile yalnız prod bağımlılık (express); build sırasında `npm audit --omit=dev --audit-level=high` (SEC-5/SEC-6). İmajda hazır gelen `node` (non-root) kullanıcısına geçiş (SEC-7); `.dockerignore` ile `.git`/`docs`/`decisions`/`tests`/`node_modules`/`pipeline-state.json` imaj bağlamından dışlanır (SEC-8). `ci.yml`'de `npm install`→`npm ci` (lockfile-deterministik) ve yeni `docker-build` job'u (push yok, yalnız Dockerfile'ın gerçekten build olduğunu doğrular).
- Değerlendirilen alternatifler: (1) `node:20-alpine` — mimari dokümanının (docs/05) orijinal varsayımı, ama Node 22 LTS ve sibling projelerle (dice-game/coinflip) tutarlılık için 22 seçildi; (2) çok-aşamalı build — derleme adımı olmadığından (statik dosya + tek prod bağımlılık) fayda sağlamıyor.
- Gerekçe: `node:22-alpine` aktif bakım/güvenlik güncellemeleri alıyor, non-root user hazır geliyor, LITE profiliyle uyumlu asgari karmaşıklık.
- Doğrulama: `docker build` başarılı; container `node src/server.js`'i hatasız başlattı (foreground doğrulama — host port bind sandbox'ta reddedildi, ama süreç seviyesinde başlangıç doğrulandı). İmaj boyutu **241MB** — ölçüldü (`docker images`).
- Riskler: **NFR-7 bütçesi (≤150MB) aşılıyor** (241MB, dice-game'de görülen 180MB'dan da yüksek) — `node:22-alpine` taban katmanı + npm katmanları bütçeyi geçiyor. Faz 12 kapı kriteri (pipeline yeşil + artefakt üretimi) bundan etkilenmiyor; Faz 15 (Bakım) teknik borcuna taşınıyor (öneri: `gcr.io/distroless/nodejs22-debian12` veya NFR-7 bütçesinin 250MB'a güncellenmesi).
- Geri alınabilirlik: Yüksek (Dockerfile/.dockerignore bağımsız dosyalar, tek satır base image değişimiyle geri dönülebilir).
- İnsan onayı: Otomatik (AUTOPILOT — imaj boyutu riski açıkça işaretlendi, kullanıcı sonradan denetler).
- Varsayım mı?: Hayır — Node 22 seçimi sibling projelerle tutarlılık, boyut sapması ölçülmüş gerçek veri.
